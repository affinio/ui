# @affino/visual-workbench

Internal Storybook workspace used for visual and interaction verification of Affino primitives.

Status: private workspace package (not published).

## Commands

```bash
pnpm --filter @affino/visual-workbench dev
pnpm --filter @affino/visual-workbench build
pnpm --filter @affino/visual-workbench test:visual
```

## Purpose

- Manual visual QA for component states and overlays.
- Regression check for interaction behavior in Storybook.
- Playground for demos and docs screenshots.
